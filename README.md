## 📌 درباره پروژه

این پروژه یک **Boilerplate** برای شروع سریع پروژه‌های Fullstack است که علاوه بر آماده بودن زیرساخت Docker، یک ساختار استاندارد و قابل توسعه نیز در اختیار شما قرار می‌دهد.

### آنچه در این پروژه آماده شده است

- 🏗️ معماری **DDD** در Backend با **NestJS**
- ⚛️ Frontend مبتنی بر **Next.js**
- 🎨 پیکربندی کامل **shadcn/ui**
- 📚 راه‌اندازی **Storybook** همراه با کامپوننت‌های اولیه
- 🐳 Docker و Docker Compose آماده Production
- 🌐 Reverse Proxy با **Caddy**
- 🐘 PostgreSQL
- 🔴 Redis
- 🔒 HTTPS خودکار
- 🛡️ Rate Limiting
- ⚙️ ساختار مناسب برای توسعه و گسترش پروژه

این ریپازیتوری به‌عنوان یک نقطه شروع (Starter Template) طراحی شده است تا بتوانید آن را Clone کرده و بدون نیاز به انجام تنظیمات اولیه، توسعه پروژه خود را آغاز کنید.
